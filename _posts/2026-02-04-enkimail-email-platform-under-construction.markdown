Today I bring you news about an exciting project that is in development: **enkimail**, a modern email sending platform with a futuristic design and robust features.

## What is enkimail?

**enkimail** is an email sending platform styled after SendGrid, Mailgun, and Brevo, but with a humbler approach and an intuitive interface. The project is a tool that is both powerful and pleasing to the eye.

### Main features

📧 **Email Management**: Complete system for sending and tracking email campaigns.

⚡ **API Ready**: REST endpoints ready for integration into your applications (in the future).

📱 **Responsive Design**: Works perfectly on desktop, tablet, and mobile.


## Technology Stack

The project is built with the best modern technologies:

```
Frontend:
✓ Next.js 16.1.6 - Modern React framework
✓ React 19 - Interactive UI
✓ TypeScript - Safe and reliable typing
✓ Tailwind CSS 4 - Modern and responsive styles

Backend:
✓ Ruby on Rails 7
✓ PostgreSQL
```
